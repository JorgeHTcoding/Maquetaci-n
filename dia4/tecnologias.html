Tecnologías

Estás son las tecnologías que habré aprendido a dominar al concluir el Boot-Camp

  • HTML: Es un lenguaje de etiquetas para la construcción de esqueletos web. (front-end)
  • JavaScript: Es el lenguaje de programación web más usado y popular del mundo. (front-end)
  • CSS: Es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado. (apoyo)
  • TypeScript: Es lo mismo que JavaScript pero con la diferencia de ser fuertemente tipado. (front-end)
  • NodeJS: Node.js es un entorno en tiempo de ejecución multiplataforma que nos permite trabajar JavaScript en el backend. (backend)
  • MySQL: MySQL es uno de los sistemas de gestión de bases de datos relacionales más usados. (back-end)
  • MongoDB: MongoDB es un sistema de base de datos NoSQL, orientado a documentos y de código abierto. (back-end)
  • API{rest}: Es una interfaz de programación de aplicaciones (API o API web) que se ajusta a los límites de la arquitectura REST. (apoyo)
