Portafolio personal


HTML - CSS


Primero proyecto enfocado a la realización de un pequeño portafolio personal el cual se irá ampliando y mejorando a medida que se avance en el curso y se vayan adquiriendo nuevos conocimientos y habilidades.

Hemos usado HTML para crear los esqueletos de los distintos ficheros usados para las diversas páginas que contiene este portafolio en conjunto con CSS el cual hemos usado para diseñar e implementar todos los aspectos gráficos y estéticos de la misma.

Puedes acceder al repositorio de este proyecto de gihub haciendo click en el icono.

Este es el primer de mucho proyectos más que están por venir y que todavía son un misterio.
